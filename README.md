# 🌍 Disaster Dash

Interactive dashboard exploring global disaster impact and humanitarian aid (2018–2024).

🔗 **Live App:** https://connect.posit.cloud/j031nich0145/content/019cee8c-36c6-62bc-9d24-6a6d646b49f5

## Setup

**Requirements:** R ≥ 4.2

Install dependencies:

```r
install.packages(c("shiny", "dplyr", "plotly", "bslib"))
```

## Running Locally

```r
shiny::runApp(".")
```

Or open `app.R` in RStudio and click **Run App**.
